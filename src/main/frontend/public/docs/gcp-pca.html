내 함정 클러스터 — 시험 직전 1순위 복습

지금까지 틀린 문제는 거의 다 아래 4개 패턴에서 나왔다. 이 페이지만 시험장 가기 전 다시 본다.

함정 1 · 네트워킹 도구 family NAT·Direct Peering·Private Google Access를 "내부 연결/인바운드 접속"에 잘못 씀.
→ SSH 들어가기 = IAP · VPC끼리 = VPC Peering · 고성능 온프렘 = Dedicated Interconnect
함정 2 · least privilege as restricted as possible 보이면 Owner·Editor·Org Admin 보기 즉시 탈락. 가장 좁은 predefined 역할 선택.
